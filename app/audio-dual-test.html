双路音频采集验证

验证 Windows Chrome 能否同时抓到「系统声音」和「麦克风」两路独立音频。不接入会议，纯本地测试。
  1. 点「开启麦克风」，允许麦克风权限，对着麦说话，看蓝色条是否跳动。
  2. 点「开启系统声音」，在弹窗里选择「整个屏幕」或某个标签页，并务必勾选左下角「分享系统音频 / Share system audio」。
  3. 播放一段有声音的视频/音乐，看绿色条是否跳动。
  4. 两条各自独立跳动 = 双路可行；绿色条始终不动 = 系统声没抓到（多半是没勾选系统音频，或共享了单个窗口）。
